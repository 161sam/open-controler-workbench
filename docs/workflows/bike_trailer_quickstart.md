# Bike Trailer Quickstart

## Goal

Create a believable first trailer concept with frame, running gear, hitch, and optional cargo volume.

## Recommended Start

1. Use `Select Domain` and choose `Bike Trailer`.
2. In the Create panel, start with one of these templates:
   - `Starter Flatbed Trailer` for an open utility base
   - `Cargo Box Trailer` for a boxed cargo hauler
3. Pick the size preset that is closest to the intended use, then create the trailer layout.

## First Useful Result

- `Starter Flatbed Trailer`: begin with frame crossbars and axle mounts, then confirm wheel stance and hitch position.
- `Cargo Box Trailer`: use the boxed template when the cargo volume matters earlier than the frame details.

## Typical Next Steps

1. Build or refine in this order:
   - `Frame Setup`
   - `Rolling Gear`
   - `Cargo Modules`
2. Use direct placement and move tools to tune wheel spacing, hitch alignment, and cargo position.
3. Refine overall width and length with template parameters if the starter preset is close but not final.
4. Keep the first pass simple: confirm the concept flow before turning it into a fully engineered trailer.
